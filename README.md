`url: https://space-x-svelte.netlify.app/`

## Instructions

1. Clone the repository.

```bash
git clone https://github.com/nihaltowfiq/space-x-svelte.git
```

2. Go to the project folder

```bash
cd space-x-svelte
```

3. Install dependencies

```bash
npm install
```

4. Run

```bash
npm run dev
```
